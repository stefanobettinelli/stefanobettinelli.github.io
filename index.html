About

I'm a software engineer with 2 years of working experience and a MSc Degree from University of Bologna. I started my professional career as a junior developer for NoemaLife, a healthcare software house based in Bologna, nice environment and filled with smart people. I then moved to a research fellowship program at CNR in Bologna to work on the development of an IoT platform for the collection of atmospheric data. My main responsibilities in the project are to work both at the front-end and back-end of the platform (mainly with Javascript, Java and Python). Currently my main goal is to improve as a software engineer to reach a senior level.

I'm also a Self-Taught Artist as I love drawing, something that I have enjoyed a lot since I was a little boy. It's my main source of relaxation, a form of meditation that allows my imagination to flow without restrictions.
That's why I like to think of myself more as visual person, I rarely go straight to code typing when it comes to problem solving or building something from skratch. Instead I firstly rely on a simple pencil and paper to sketch diagrams, UI mockups etc. It is inspiring working on front-end development. I love watching how people interact with the tools and the UIs I'm able to create. Last but not least I'm really proud that I have some game development experience under my belt, watching people playing at my games is such a fun and satisfying experience.

Location

Currently based in Bologna (Italy)

Around the Web

Air pollution platform Data UI

At CNR after building the backend stack (I was specifically in charge of building and maintaining the REST API using Spring Boot), my data scientist colleagues needed a simple UI tool for viewing the data collected by our platform and also a simple and easy way to download them in CSV format. To build this UI I've used React along with Redux for state managment and material-ui as a UI framework.
